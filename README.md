# Assignment 1

## Title

Analysis of Time Complexities: Constant Time, Linear Time and Quadratic Time

## Question

Write a C program to analyze time complexities in constant time, linear time and quadratic time. Increase the size of input to check the time consumed.

## Objective

To understand and compare the execution time of algorithms with different time complexities:

* Constant Time O(1)
* Linear Time O(n)
* Quadratic Time O(n²)

## Description

The program measures the time taken by three different operations representing constant, linear and quadratic time complexities. By increasing the input size, the effect of each complexity on execution time can be observed and analyzed.

## Language Used

C Programming

## Files

* `time_complexity.c`

## Compilation and Execution

```bash
gcc time_complexity.c -o output
./output
```

## Expected Observation

* O(1) execution time remains nearly constant regardless of input size.
* O(n) execution time increases proportionally with input size.
* O(n²) execution time increases rapidly as input size grows.
