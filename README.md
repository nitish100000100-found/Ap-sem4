# Assignment 1

## Question

Write a C program to analyze time complexities in constant time, linear time and quadratic time. Increase the size of input to check the time consumed.

## Program Description

This program demonstrates and compares the execution time of:

* Constant Time O(1)
* Linear Time O(n)
* Quadratic Time O(n²)

The program measures the time taken for each operation using different input sizes and displays the results.
